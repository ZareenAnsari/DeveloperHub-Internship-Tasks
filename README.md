# DeveloperHub Internship Projects

This repository contains the machine learning tasks completed during my internship.

## 1. Bank Customer Churn Prediction
*   **Objective:** Predict which customers are likely to leave the bank to help improve retention.
*   **Approach:** Cleaned data and handled imbalances. I then applied **[* Gaussian Naive Bayes
    * Logistic Regression
    * Decision Tree
    * Random Forest
    * Extra Gradient Boosting Tree (XGBoost)] for classification.
*   **Results:** Achieved **[ 90%]** accuracy. I found that **[Credit Score]** was a major factor in churn.

## 2. Personal Loan Acceptance Prediction
*   **Objective:** Identify potential customers who will accept personal loan offers.
*   **Approach:** Conducted Exploratory Data Analysis (EDA) and built a model using **[Logistic Regression]**.
*   **Results:** Identified that income level and education were the strongest predictors for loan acceptance.

## 3. Predicting Insurance Claim Amounts
*   **Objective:** Estimate the cost of insurance claims based on user data for better financial planning.
*   **Approach:** Performed regression analysis using **[Linear Regression]**.
*   **Results:** The model achieved an R² Score of 0.80, indicating that it explains 80% of the variance in claim amounts, with a Mean Squared Error (MSE) of 0.18.
